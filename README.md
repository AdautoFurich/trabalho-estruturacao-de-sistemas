# ESTRUTURAÇÃO DE SISTEMAS E APLICAÇÕES

Este pacote contém os projetos:
- `exercicio1_basico`
- `exercicio1_mvc`
- `exercicio2_basico`
- `exercicio2_mvc`
- `exercicio3_basico`
- `exercicio3_mvc`

Cada projeto possui:
- `README.md` com tabelas e regras de negócio identificadas;
- código Java com classes e relacionamentos;
- classe `Main` simulando pelo menos um movimento do sistema.

## Estrutura adotada
Para atender ao pedido de “2 projetos para cada exercício”, foram criadas duas versões:
1. **Versão Básica**: classes e relacionamentos diretos.
2. **Versão MVC**: separação em `model`, `controller` e `view`.
