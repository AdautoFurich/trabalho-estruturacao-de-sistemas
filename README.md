# 📚 Trabalho de Estruturação de Sistemas

Projeto desenvolvido para a disciplina de Estruturação de Sistemas, com o objetivo de aplicar conceitos de análise, modelagem e desenvolvimento de software utilizando boas práticas de engenharia de software.

O trabalho busca demonstrar a aplicação prática dos conhecimentos adquiridos durante o curso de Análise e Desenvolvimento de Sistemas.

---

## 🎯 Objetivo

Desenvolver uma solução computacional aplicando conceitos de:

* Levantamento de requisitos
* Modelagem de sistemas
* Estruturação de software
* Organização de componentes
* Documentação técnica
* Desenvolvimento orientado a objetos

---

## 🏗️ Escopo do Projeto

O sistema foi desenvolvido para atender aos requisitos propostos na disciplina, contemplando as principais etapas do ciclo de desenvolvimento de software.

Entre elas:

* Análise do problema
* Definição dos requisitos
* Modelagem do sistema
* Implementação da solução
* Validação dos resultados

## ⚙️ Como Executar

Clone o repositório:

```bash
git clone https://github.com/AdautoFurich/trabalho-estruturacao-de-sistemas.git
```

Entre na pasta:

```bash
cd trabalho-estruturacao-de-sistemas
```

Abra o projeto em sua IDE de preferência:

* IntelliJ IDEA
* Eclipse
* VS Code (Java Extension Pack)

Execute a classe principal:

```java
Main.java
```
